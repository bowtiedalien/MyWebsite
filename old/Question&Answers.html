Q: What exactly is bash in Linux?
A: bash is a file to process the commands that you can run in the terminal window. It allows you to write commands and can read commands from an input file! So you can think of it as a translation service between you as the programmer and the operating system.

Q: How can I install a Linux distribution from a USB?
A: Ok, this is how I did it:
  1. install the .iso image of the Linux distribution you want to use. Write the ditribution's name in Google and then "install" and you'll get links to install the .iso file. For beginners, Ubuntu or Linux Lite can be used.
  2. burn this .iso image to a USB or DVD. I used a USB. A software that can help you make the USB a bootable device (meaning: so that Windows can identify that there is an operating system on it that can be booted) is Rufus. You can find how exactly to use Rufus to convert a USB to a bootable USB from this link:
    Create bootable USB using Rufus
  3. Now you have to tweak the BIOS settings. To access BIOS settings, you can search "Advanced startup" in Windows 10 and click "Restart Now". If that doesn't work, search Google on how to access BIOS on your computer and operating system. Anyway, I changed a bunch of settings
    • ENABLE legacy
    • DISABLE secure boot
    • edit boot order from boot menu so that the USB boots first
    • additional step: after doing all this, I still couldn't run the bootable file from the USB because it seemed as if Windows was just ignoring it as if it didn't exist. Later I found out this was due to the "fast boot" setting that Windows had. So I disabled that. And because the boot menu was not showing automatically, I also had to change the startup delay from 0 to 10 seconds so that I had time to click ESC to move the boot menu manually. This allowed me to see all the possible options that I could boot from and I was able to choose my USB.
  4. now the USB will boot and you will be greeted with a welcome screen and an option to install the Linux distro. Do that. If you're trying to install Linux Lite, use this link: Install Linux Lite
    Otherwise, search on how to install your chosen Linux distro.
